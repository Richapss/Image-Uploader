Example Code

Copy dan gunakan kode berikut untuk mengintegrasikan fitur upload gambar ke aplikasimu.

Upload Endpoint http://localhost:3001/upload
JavaScript (Fetch) js 
const uploadImage = async (file) => {
  const fd = new FormData();
  fd.append("image", file);

  const res = await fetch("http://localhost:3001/upload", {
    method: "POST",
    body: fd,
  });

  const data = await res.json();
  if (!res.ok) throw new Error(data.error || "Upload gagal");

  return data.url;
};
JavaScript (Axios + Buffer) js 
const uploadImage = async (buffer, ext = "jpg", mime = "image/jpeg") => {
  const fd = new FormData();
  fd.append("image", new Blob([buffer], { type: mime }), `file.${ext}`);

  const res = await axios.post("http://localhost:3001/upload", fd, {
    maxBodyLength: 50 * 1024 * 1024,
    maxContentLength: 50 * 1024 * 1024,
  });

  const data = res.data || {};
  if (!data.url) throw new Error("Upload gagal atau response tidak valid");

  return data.url;
};
cURL bash 
curl -X POST http://localhost:3001/upload \
  -F "image=@/path/to/gambar.jpg"

Response akan berisi url — link unik gambar yang bisa langsung dibagikan. Gambar otomatis terhapus jika tidak diakses selama 14 hari.
